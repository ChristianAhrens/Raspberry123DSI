# Documentation

Welcome to the Raspberry123DSI documentation. This site contains guides for building the enclosure, configuring the Raspberry Pi OS, and setting up the audio routing.

## Files

All raw configuration files are available directly in the [config/](https://github.com/ChristianAhrens/Raspberry123DSI/tree/main/config) directory of the repository.

All laser-cut SVG files are in the [rack_enclosure/](https://github.com/ChristianAhrens/Raspberry123DSI/tree/main/rack_enclosure) directory and downloadable from the [assets page](https://ChristianAhrens.github.io/Raspberry123DSI/assets/).
